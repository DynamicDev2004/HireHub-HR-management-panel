HireHub – HR Management Panel
HireHub is a web-based HR management system developed using C#, ASP.NET, SQL Server, and Tailwind CSS. The platform streamlines recruitment, employee management, and hiring workflows for organizations.

• Features

• HR/Admin Panel
Create, update, and manage job listings
Manage employees (hire, remove, update details)
Maintain salary records
Track candidates through a 3-stage interview process
Update interview status visible to applicants
Centralized admin dashboard

• User / Candidate Panel
View available job openings
Apply for jobs online
Upload CV/Resume
Track interview progress across all stages
View application and interview status in real time

• Tech Stack
Backend: C#, ASP.NET (.NET)
Frontend: Tailwind CSS
Database: SQL Server
Architecture: Role-based access (HR & Candidates)

• Purpose
HireHub is designed to simplify recruitment by providing transparency for candidates and full control for HR teams, all within a modern and scalable system.
